# CSharp OOP Inheritance

This project demonstrates Object-Oriented Programming (OOP) concepts in C#, 
with a focus on **Inheritance**, **Interfaces**, and **Polymorphism**.

## Concepts Covered
- Inheritance
- Base and Derived Classes
- Method Overriding
- Interfaces
- Polymorphism

## Project Structure
- `Vehicle` → Base class
- `Car` → Derived class from Vehicle
- `Shape` → Base class
- `Circle` → Derived class from Shape
- `IPrintable` → Interface example
- `IBorrowable` → Interface example
- `Program.cs` → Main entry point with test cases

## Technologies Used
- C#
- .NET Console Application

## Purpose
This project is created as a practice for understanding and applying 
OOP Inheritance concepts in C#.
